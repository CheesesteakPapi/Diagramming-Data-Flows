# Diagramming-Data-Flows

This repository contains Mermaid diagrams for various data flows and processes.

## Training Operations Flow

The training operations flow diagram illustrates the complete workflow for:
- **Admin Operations**: Course and session creation with configurable requirements
- **Registrar Check-in**: Attendee registration and check-in process
- **Device Issuance**: ITFO device assignment for permanent and TDY personnel
- **Axon Automation**: Bulk user creation and account management
- **Returns & Closeout**: Device returns and compliance tracking

View the diagram: [training-operations-flow.mmd](training-operations-flow.mmd)

### Rendering the Diagram

You can render this Mermaid diagram using:
- GitHub's native Mermaid support (in markdown files)
- [Mermaid Live Editor](https://mermaid.live/)
- VS Code extensions (e.g., Markdown Preview Mermaid Support)
- Any other Mermaid-compatible renderer
